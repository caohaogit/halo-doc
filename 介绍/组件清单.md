## 组件清单

| :--- | :--- | :--- |
| `spring-framework` | `halo-common` |  |

| `FastJSON` | `halo-common` |  |
| `apache-commons` | `halo-common` |  |
| `slf4j` | `halo-log` |  |
| `logback` | `halo-log` |  |
| `kafka-clients` | `halo-log`, `halo-kafka` |  |
| `Jedis` | `halo-cahce` |  |
| `Shiro` | `halo-auth` |  |
| `CXF` | `halo-cxf` |  |
| `spring-jdbc` | `halo-dao` |  |
| `Druid` | `halo-dao` |  |

| `Quartz` | `halo-batch` |  |
| `spring-batch` | `halo-batch` |  |
| `curator-framework` | `halo-tune` |  |
